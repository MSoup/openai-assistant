# openai-assistant

Playing with openai API

# Prerequisites

-   Python 3.11+

# Getting started

-   Make a virtual env

```bash
python3 -m venv venv
```

-   Install dependencies

```bash
pip install -r requirements.txt
```

-   Populate env with your own openai API key

```bash
echo OPENAI_API_KEY=YOUR_API_KEY > .env
```
